# Prompt-to-Video
Official implementation of Prompt-to-Video.
![p2v_images](p2v_images.png)

## Installation
For install and running this repo:

1. Install requierd packages:
```
Python3 -m pip install requirements.txt
```

2. Run bash file:
 ```
bash install.sh
```
